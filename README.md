# NoMoreScamTW

**NoMoreScamTW** 是一個 Chrome 瀏覽器擴充功能，旨在保護使用者免受詐騙網站的侵害。它會自動比對政府公開的詐騙網站清單，當您瀏覽到可疑網站時，即時跳出紅色警示視窗，防止個人資料或財產受損。

## ✨ 主要功能

- **即時防護**：當瀏覽到詐騙網站時，自動彈出全螢幕紅色警示。
- **每日自動更新**：系統會每 24 小時自動從政府資料開放平臺下載最新的詐騙網站清單。
- **智慧啟動檢查**：若電腦關機導致錯過更新時間，擴充功能會在下次啟動瀏覽器時立即檢查並更新資料庫。
- **資訊透明**：彈出視窗會顯示該詐騙網站的類型或名稱（例如：xx娛樂城、假冒購物網）。
- **隱私優先**：所有的網址比對皆在**本地端 (Local)** 進行，您的瀏覽紀錄**不會**被傳送至任何伺服器。

## 📥 安裝說明

你可以選擇從 Chrome 線上應用程式商店安裝，或是手動載入原始碼版本：

### 1. 官方商店安裝（推薦）
請至 [Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/emhndljhamhamjilimolbbinpipbkjlh) 點擊 **「加到 Chrome」**。

### 2. 手動安裝（開發者版本）
1.  下載此專案原始碼。
2.  開啟 Google Chrome 瀏覽器，在網址列輸入 `chrome://extensions` 並按下 Enter。
3.  在右上角開啟 **「開發人員模式」 (Developer mode)**。
4.  點擊左上角的 **「載入未封裝項目」 (Load unpacked)** 按鈕。
5.  選擇此專案的資料夾。
6.  安裝完成！您會在擴充功能列看到一個藍色的盾牌圖示。



## 🛡️ 使用方法

1.  **檢查狀態**：點擊瀏覽器右上角的盾牌圖示，可查看目前資料庫的更新時間及網址數量。
2.  **手動更新**：若需立即獲取最新資料，可點擊 Popup 視窗中的「更新資料庫」按鈕。
3.  **遇到警示時**：
    - 點擊 **「立即離開 (回到 Google)」**：安全地離開詐騙網站。
    - 點擊 **「我了解風險，繼續瀏覽」**：關閉警示視窗 (僅供測試或誤判時使用)。
