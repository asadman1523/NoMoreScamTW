# 隱私權政策 (Privacy Policy)

**NoMoreScamTW** (以下簡稱「本擴充功能」) 非常重視您的隱私。本政策旨在說明我們如何處理您的資料。

## 1. 資料收集與使用 (Data Collection and Use)

本擴充功能 **不會** 收集、儲存或傳送您的任何個人資料、瀏覽紀錄或使用者活動至任何外部伺服器。

*   **網址比對**：所有的網址比對作業皆在您的電腦本機端 (Client-side) 進行。
*   **黑名單下載**：本擴充功能僅會定期從政府資料開放平臺 (https://data.gov.tw/) 下載最新的公開詐騙網站清單。

## 2. 權限說明 (Permissions)

為了提供防護功能，我們需要以下權限，但僅用於功能運作，絕無資料外洩之虞：

*   **Storage**: 用於在您的電腦上儲存下載的黑名單資料庫。
*   **Tabs / Web Navigation**: 用於偵測您當前瀏覽的網址，以即時比對是否為詐騙網站。
*   **Alarms**: 用於設定每日自動更新黑名單的排程。

## 3. 第三方服務 (Third-Party Services)

本擴充功能未串接任何用於追蹤、分析或廣告的第三方服務 (如 Google Analytics)。

## 4. 聯絡我們 (Contact)

若您對本隱私權政策有任何疑問，歡迎透過 GitHub Issue 聯繫我們。
